Propósito de esta página web

Este sitio tiene web como propósito, fomentar el aprendizaje en las temáticas correspondientes Flex Box y CSS, es más fácil personalizar los diseños poder alterar el ancho, alto y posicionamiento de elementos de la mejor manera con el espacio del que disponemos. Con esta herramienta podemos dar tamaño las filas y las columnas, añadiendo las preferencias al contenedor con el fin de que se les haga más fácil seguirlos estándares de diseño, para que el contenido se muestre de una manera agradable y sea de fácil comprensión para quienes lo utilicen.

Introducción

Cuando se habla de FlexBox o CSS Grid, nos referimos a herramientas de maquetación para nuestraspáginas web, estas herramientas las podemos encontrar cuando diseñamos en HTML5, ahora bien, Flexbox se creó para diseños de una sola dimensión, es decir, una fila o una columna, podríamos decir queCSS Grid es la evolución de este método porque nos permite trabajar con varias filas y varias columnas de manera simultánea.
